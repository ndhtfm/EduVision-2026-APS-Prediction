# EduVision 2026: APS Prediction Project

EduVision 2026 is a data mining project that predicts the School Participation Rate (SPR) or Angka Partisipasi Sekolah (APS) for individuals aged 19–23 in Indonesia.

## Project Description

This project predicts the 2026 APS for the 19–23 age group based on provincial socioeconomic indicators in Indonesia. The data used in this project were obtained from Statistics Indonesia (BPS) for the period 2020–2025.

## Variables Used

The predictor variables are:

- Adjusted per capita expenditure
- GRDP per capita
- Human Development Index (HDI)
- Number of poor people
- Percentage of poor people
- Open unemployment rate

The target variable is:

- School Participation Rate (SPR/APS) aged 19–23

## Methods

This project uses two prediction models:

- Linear Regression
- Random Forest Regressor

The models are evaluated using:

- MAE
- RMSE
- R-squared

## Main Result

Random Forest Regressor was selected as the main model because it produced the lowest MAE.

The predicted average APS for 2026 is 31.90%, increasing from the 2025 average of 29.48%.
